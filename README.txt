Steps to run the project :

1. Copy the project finalprojectserver from "https://github.com/solka03/finalprojectserver/tree/develop" - Development branch

2. Go to the project directory and run the command "docker build -t uuid-server"

3. Check the container "uuid-server" is available using "docker ps" command

4. Copy the project finalprojectclient from "https://github.com/solka03/finalprojectClient/tree/develop" - Development branch

5. Go to the project directory and run the command "docker build -t uuid-client"

6. Check the container "uuid-client" is available using "docker ps" command

7.Now run the docker-compose.yml file using command "docker-compose up"

8.Check the output on command prompt
